// Write-up file format

Each write-up lives in a Writeups/ subfolder as its own HTML file. Recommended naming: YYYY-MM-CTFNAME-challenge-name.html

To add a new write-up, add an entry to Writeups/index.json:

  • title — challenge name shown in the table
  • event — CTF event name
  • category — one of: pwn / re / web / forensics / crypto
  • date — e.g. Mar 2026
  • file — path relative to root, e.g. Writeups/2026-03-ctf-challenge.html
